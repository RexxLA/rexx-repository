Methods

  • Activate CLASS

    The activate class method is automatically called at the end of the class construction. It creates two stems, encode. and decode., that will be used by the encode and decode methods.

    See https://www.unicode.org/Public/MAPPINGS/VENDORS/MICSFT/PC/CP850.TXT for details.

  • decode CLASS

    CP850 decoding always succeeds (i.e., there is no such thing as a badly encoded cp850 sequence).

    USE ARG string, format = "", errorHandling = ""
  • encode CLASS

    CP850 encoding will fail, unless the argument string contains only characters that can be represented in CP850.

    USE ARG string, option = ""
